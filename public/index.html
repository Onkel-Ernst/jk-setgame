MultiplayerSet

  • Set ist ein Spiel, in dem man SETS aus drei Karten auf einem Spielfeld mit zwölf Karten finden muss. Welche Karten zu einem SET gehören, hängt mit ihren VIER Eigenschaften ab, welche jede Karte besitzt:

    • Form: oval, Karo, or Kringel
    • Farbe: rot, blau oder grün
    • Anzahl der Formen: eine, zwei oder drei
    • Muster: gefüllt, hohl oder gestreift

  • Sets

  • Ein SET besteht aus drei Karten, bei denen alle Eigenschaften auf den drei Karten entweder alle gleich sind…

    oder alle unterschiedlich…

    oder eine Mischung aus gleichen und verschiedenen Eigenschaften. Hier haben die Karten Anzahl und Form gemeinsam bei komplett unterschiedlichen Farben und Mustern.

  • Faustregel

  • Wenn man sich nicht sicher sind, ob man ein SET gefunden hat, fragt man sich: "Gibt es nur zwei von jeder Eigenschaft?" Wenn ja, dann hat man kein SET. Hier sind einige Beispiele:

    Dies wäre ein SET, außer dass zwei der Karten blau sind. Wenn sie alle blau wären oder jede Farbe anders, so hätte man ein richtiges SET.

    Gleiches Problem hier: zwei der Karten haben gefülltes Muster. Wenn alle drei gefüllt wären oder alle unterschiedliche Muster hätten, dann wäre es korrekt.

  • Spiel starten

  • Um ein Spiel zu beginnen, wählt man Start in der linken oberen Ecke. Man kann den angezeigten Link des aktuellen Spielraumes nutzen, um einen Freund einzuladen (bis zu acht Spieler).

    Klicke auf Karten, um ein SET zu bilden. Eine Auswahl kann aufgehoben werden indem man auf eine beliebige Stelle des Spielbretts klickt.

    Tipps

    Wenn du einen Hinweis brauchst oder kein SET auf der Tafel siehst (was passieren kann), benutze die Schaltfläche TIPP. Wenn die Mehrheit der Spieler dies tut, wird eine Karte eines SETs auf dem Brett enthüllt. Wenn es keine gibt, werden drei zusätzliche Karten ausgeteilt.
